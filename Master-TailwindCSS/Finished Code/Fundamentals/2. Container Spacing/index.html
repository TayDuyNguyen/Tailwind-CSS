6. Responsive Spacing - Spacing responsive

Thay đổi kích thước trình duyệt để xem responsive

Responsive Margin: m-2 (mobile) → m-4 (tablet) → m-8 (desktop) → m-12 (large)
Responsive Padding: p-2 → p-4 → p-8 → p-12
Container với responsive padding: px-4 → px-6 → px-8
